Cómo iniciar el proyecto con Vite

Para iniciar un proyecto con Vite, React y TypeScript, puedes usar los siguientes comandos:

npm create vite@latest my-app --template react-ts cd my-app npm install npm run dev

Para instalar Zustand y utilizarlo en tu proyecto:

npm install zustand
